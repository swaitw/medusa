# updateProductCategoriesWorkflow - Medusa Core Workflows Reference

This documentation provides a reference to the `updateProductCategoriesWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow updates product categories matching specified filters. It's used by the
[Update Product Category Admin API Route](https://docs.medusajs.com/api/admin#product-categories\_postproductcategoriesid).

You can use this workflow within your customizations or your own custom workflows, allowing you to
update product categories within your custom flows.
