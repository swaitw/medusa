# getExistingPriceListsPriceIdsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `getExistingPriceListsPriceIdsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step retrieves prices of price lists.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { getExistingPriceListsPriceIdsStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = getExistingPriceListsPriceIdsStep({
      "price_list_ids": [
        "{value}"
      ]
    })
  }
)
```
