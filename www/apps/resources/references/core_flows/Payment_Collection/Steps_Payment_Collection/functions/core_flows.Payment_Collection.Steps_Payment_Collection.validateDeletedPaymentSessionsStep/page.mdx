# validateDeletedPaymentSessionsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateDeletedPaymentSessionsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that the specified payment session IDs were deleted.
If not all payment sessions were deleted, the step throws an error.

## Example

```ts
const data = validateDeletedPaymentSessionsStep({
  idsDeleted: ["pay_123"],
  idsToDelete: ["pay_123"]
})
```
