# batchLinkProductsToCollectionStep - Medusa Core Workflows Reference

This documentation provides a reference to the `batchLinkProductsToCollectionStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step manages the links between a collection and products.

```ts
const data = batchLinkProductsToCollectionStep({
  id: "collection_123",
  add: ["product_123"],
  remove: ["product_321"]
```
