# registerOrderChangesStep - Medusa Core Workflows Reference

This documentation provides a reference to the `registerOrderChangesStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step registers an order changes.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { registerOrderChangesStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = registerOrderChangesStep({
      "order_id": "order_123"
    })
  }
)
```
