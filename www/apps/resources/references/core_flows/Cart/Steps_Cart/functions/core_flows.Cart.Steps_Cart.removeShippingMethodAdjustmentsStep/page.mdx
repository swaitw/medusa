# removeShippingMethodAdjustmentsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `removeShippingMethodAdjustmentsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step removes shipping method adjustments from a cart.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { removeShippingMethodAdjustmentsStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = removeShippingMethodAdjustmentsStep({
      "shippingMethodAdjustmentIdsToRemove": [
        "{value}"
      ]
    })
  }
)
```
