# createDefaultSalesChannelStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createDefaultSalesChannelStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates a default sales channel if none exist in the application. 
This is useful when creating seed scripts.

## Example

```ts
const data = createDefaultSalesChannelStep({
  data: {
    name: "Webshop",
  }
})
```
