# removeRulesFromPromotionsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `removeRulesFromPromotionsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step removes rules from a promotion.

## Example

```ts
const data = removeRulesFromPromotionsStep({
  rule_type: RuleType.RULES,
  data: {
    id: "promo_123",
    rule_ids: ["prule_123"]
  }
})
```
