# addCampaignPromotionsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `addCampaignPromotionsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step adds promotions to a campaign.

## Example

```ts
const data = addCampaignPromotionsStep({
  id: "camp_123",
  add: ["promo_123"],
```
