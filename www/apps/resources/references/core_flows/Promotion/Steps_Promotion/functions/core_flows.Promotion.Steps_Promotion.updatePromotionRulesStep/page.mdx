# updatePromotionRulesStep - Medusa Core Workflows Reference

This documentation provides a reference to the `updatePromotionRulesStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step updates one or more promotion rules.

## Example

const data = updatePromotionRulesStep({

      id: "prule_123",
