# createInventoryLevelsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createInventoryLevelsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates one or more inventory levels.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { createInventoryLevelsStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = createInventoryLevelsStep({
      "inventory_item_id": "{value}",
      "location_id": "{value}"
    })
  }
)
```
